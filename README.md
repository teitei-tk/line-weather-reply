1: fetch ChannelSecret and ChannelToken

    * https://business.line.me/en/services/bot

2: exec code
```bash
$ export CHANNEL_SECRET=test CHANNEL_TOKEN=foo; go run main.go
```

3: add friend a bot  
4: send a appropriate message to your bot
